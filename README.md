# file-manager

### vim-like file manager written in c

To run:
```bash
./init
```
```bash
make && make run
```

## Controls
* Vertical arrow keys to switch throw dirs
* commands:
	* add file/folder
		```
		:a <file>
		```
		```
		:a <folder>/
		```
	* remove file/folder
		```
		:r <file>
		```
		```
		:r <folder>/
		```
