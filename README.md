## croco - 🐊Efficient terminal file manager  

🖥️ Start script for smart initialization:
```bash
./init
```
Or you can just make it 🤓
```bash
make && make run
```

## Controls
* Arrow keys || h-j-k-l to navigate through dirs
* Enter to open the file / enter the directory
* commands:
	* add file/folder
		```
		:a <file>
		```
		```
		:a <folder>/
		```
	* remove file/folder
		```
		:r <file>
		```
		```
		:r <folder>/
		```
